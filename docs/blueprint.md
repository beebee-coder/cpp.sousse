# **App Name**: VisioNode Control

## Core Features:

- Intelligent Vision Assistant: An AI-driven assistant that uses tool-based reasoning to analyze visual feeds via Groq and Gemini APIs for object detection and categorization.
- Hybrid Pipeline Controller: Smart context detection to toggle between local Sharp processing on Desktop and API-driven cloud processing for the Web client.
- Visual RAG Registry: Cloud-based vector search implementation using Pinecone and Neon PostgreSQL to retrieve industrial technical documents based on visual input.
- Live Event Hub: Real-time state synchronization using Pusher and Upstash Redis for multi-platform status updates across web and desktop instances.
- Universal Video Terminal: Unified dashboard interface supporting local industrial USB camera feeds and standard browser media streams with edge runtime efficiency.
- Vercel Blob Media Vault: Chunked image upload and storage system optimized for large industrial image files within Vercel serverless constraints.
- Vision Slash Command Palette: Interactive command bar for executing complex visual tasks, indexing images, and triggering AI queries quickly within the UI.

## Style Guidelines:

- A high-precision dark theme with an industrial 'Precision Cyan' primary color (#32b5d4) set against a 'Deep Carbon' background (#101517).
- An accent 'Terminal Green' (#2eb892) used for status indicators, mimicking high-tech dashboard instrumentation.
- Headlines feature 'Space Grotesk' for a computerized, scientific feel; body text utilizes 'Inter' for objective clarity; logs use 'Source Code Pro'.
- A rigid, modular grid system that feels like a professional engineering cockpit, using condensed shadcn/ui components.
- Clean, technical line icons with thin strokes to emphasize the application's analytical and precise nature.
- Linear and geometric transitions that convey high-performance processing without decorative flourish.
